### PROCESO DE INSTALACION
Instala archivos necesarios
```bash
- pkg upgrade && update
- pkg install nodejs
- pkg install git
- pkg install bash
```
Clona este repositorio
 ```bash
> git clone https://github.com/thechoute/BabyBechoso
```
Inicia la instalacion
```bash
- cd BabyBechoso
- bash install.sh
```
PARA INICIAR EL BOT

 ```bash
- npm start
```
